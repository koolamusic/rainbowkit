# Compact Mode

## Default vs. Compact Mode

By default, Rainbowkit's layout looks like this:

<Img src="/hero-modal.png" layout="fill" objectFit="contain" />

We also provide a `compactMode` option, which looks like this:

<Img src="/hero-modal.png" layout="fill" objectFit="contain" />

To enable Compact Mode, just include the `compactMode` prop in your `RainbowKitProvider`:

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider compactMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```
